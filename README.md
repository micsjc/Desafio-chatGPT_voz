# Desafio-chatGPT_voz
Implementa o desafio de utilizar o chatGPT, junto com speech-to-text e text-to-speech.
Este projeto implementa um menu por voz, em que o usuário é solicitado a escolher uma opção, entre as disponíveis, por voz. Essa entrada é enviada para o chatGPT, que seleciona entre as opções disponíveis, retornando uma mensagem bem humorada caso não entenda. Isso faz com que o usuário não precise ser exato em sua resposta.
Por fim, a solicitação é executada.
Opções implementadas:
1- Gravar nota em áudio
2- Gravar nota em texto
3- Ouvir última nota de áudio
4- Ouvir última nota de texto
